== Description ==
Flickr Instagram Picasa Gallery is an awesome gallery plugin to display your Flickr/Instagram/Picasa Albums on your WordPress blog.
Check Quick Video <a href="https://www.youtube.com/embed/cnILevtzYN8">Here</a>

<h3><strong>Key Features & Options</strong></h3>
<ul>
<li>Beautiful Gallery</li>
<li>Lightbox</li>
<li>Get the image from flicrk ID</li>
<li>Get the image from Instagram user id</li>
<li>Get the image from Instagram tag</li>
<li>Get the image from Instagram popular</li>
<li>Get the image from Picasa</li>
<li>Free supports.</li>
</ul>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
1. Go to the menu 'Plugins' -> 'Install' and search for 'Flickr Instagram  Picasa Gallery'
2. Click 'install'.
3. Go to Post/Page or any custom post type (support editor) and click on shortcode, folow the screenshot below to know more
http://test.wiloke.com/wp-content/uploads/2015/03/screenshot-2.png
